# ARINC_RDSA

# Dependencies
- C++17 MSVC compiler
- Qt >= 6.5
- mlpack >= 4.5.1
- MySQL Connector >= 9.1

# Building from source
To build from source you will need CMake
- clone repository
- set Qt6_DIR, Armadillo_DIR, BLAS_LIBRARIES, LAPACK_LIBRARIES, MYSQL_INCLUDE_DIRS, MYSQL_LIBRARIES, MLPACK_INCLUDE_DIRS
- build solution using CMake
